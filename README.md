# ニコニコライエジョフ

> ニコニコ動画人民委員、ニコニコライエジョフ。

LLM の tool use でニコニコ動画のコメントをAIフィルターするユーザースクリプト。

名称は「ニコニコ」と、ソ連の NKVD 長官ニコライ・エジョフ (Nikolai Yezhov) のかばん語。
AI人民委員がコメント欄の治安を守り、不適切なコメントを粛清する。

## なぜ作ったか

ニコニコ動画のコメントは視聴体験の核だが、荒らし・煽り・ネタバレで台無しになることがある。
既存のNGワード機能では文脈を理解できず、あらゆるパターンを事前に登録するのは不可能。

コンテンツが無限に増え続け、参加者も増え続ける世界では、
「文脈を理解して自然言語で指示できるフィルター」が必要になる。
50年後にはこれが当たり前になっているはず — ニコニコライエジョフはそのプロトタイプ。

## 概要

ニコニコ動画プレイヤーの内部構造 (React Fiber, Immer ストア, PixiJS レンダラー) に直接アクセスし、
チャットインターフェースでAI人民委員と対話しながらコメントの非表示・書き換えをリアルタイムに実行する。

### 対応プロバイダー

| プロバイダー | モデル例 | 1回あたりのコスト目安 |
|---|---|---|
| Anthropic (Claude) | Haiku 4.5, Sonnet 4.6, Opus 4.6 | ¥1〜10 |
| Google (Gemini) | Flash-Lite, 3.5 Flash, 3.1 Pro | ¥0.2〜5 |
| OpenAI (GPT) | GPT-5.4 nano/mini/5.4, GPT-5.5 | ¥0.2〜10 |

## インストール

1. [Tampermonkey](https://www.tampermonkey.net/) をブラウザにインストール
2. **[ニコニコライエジョフをインストール](https://github.com/106-/niconikolai-yezhov/raw/refs/heads/main/niconico-comment-filter.user.js)** をクリック（Tampermonkey が自動でインストールダイアログを表示します）
3. 利用したいプロバイダーの API キーを取得:
   - [Anthropic](https://console.anthropic.com/)
   - [Google AI Studio](https://aistudio.google.com/)
   - [OpenAI](https://platform.openai.com/)

## 使い方

### 初回設定

1. ニコニコ動画の視聴ページを開く
2. プレイヤー右下の **漏斗アイコン** をクリック
3. **設定タブ** で API キーを入力（使用するプロバイダーのみでOK）
4. モデルを選択して「保存」

### コメント分析

1. **チャットタブ** の「▶ コメント N 件を分析」ボタンをクリック
2. AI が天気予報形式でコメント欄の治安を評価
3. フィルター提案がボタンとして表示される — 複数選択可
4. 「選択した N 件を実行」→ AI がコメントを即座に粛清

### チャットで対話

分析後、チャット欄から追加の指示を自然言語で出せる:

| 指示例 | 効果 |
|---|---|
| 荒らしコメントを消して | 荒らし・煽りコメントを非表示 |
| 英語コメントを日本語に翻訳して | 英語コメントを日本語に書き換え |
| ネタバレっぽいコメントを消して | ネタバレコメントを非表示 |
| この動画のコメントの傾向を教えて | コメント内容の分析・要約 |

弾幕（🍩🍩🍩、888888）、空耳、コメントアートなどニコニコ文化のコメントは保護される。

### コスト確認

**コストタブ** で累計トークン数・コスト（USD / 円換算）・動画ごとの利用履歴を確認できる。

## 技術スタック

- **対象**: ニコニコ動画 `nvpc_next` プレイヤー
- **LLM**: Anthropic / Google Gemini / OpenAI API + tool use (Function Calling)
- **API通信**: `GM_xmlhttpRequest` (CORS 回避)
- **コメント操作**: React Fiber BFS → Immer ストア + PixiJS レンダラー直接書き換え
- **永続化**: Tampermonkey `GM_setValue` / `GM_getValue`
- **デザイン**: ニコニコ公式 Panda CSS デザイントークン準拠

## 50年後のフィルター

NGワード → 機械学習分類器 → LLM tool use。フィルター技術は進化し続けている。

50年後には、各ユーザーが自分専用のAIフィルターを持ち、100言語のコメントがリアルタイムに翻訳・フィルターされ、文化的なコメントは守られつつ荒らしだけが消える — そんな世界が来るかもしれない。

ニコニコライエジョフは、その未来の原型。
